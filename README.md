Project Name :

Keen Keeper – Your Personal Relationship Manager

Project Description:

Keen Keeper is a personal relationship management web application designed to help users track and maintain meaningful connections with their friends. It allows users to monitor interaction frequency, identify friends who need attention, and efficiently manage their social circle to ensure stronger and more consistent relationships.


Technologies Used:
- React.js
- React Router
- Tailwind CSS
- daisyUI
- JavaScript (ES6)
- React Icons (Fa / Fi)
- Recharts (for data visualization)


Key Features:

📊 Friendship Analytics Dashboard with interactive Pie Chart (Call, Text, Video) using Recharts
👥 Friend Management System with detailed friend profiles and relationship tracking
🔍 Advanced Search & Sorting to quickly find and organize friends
